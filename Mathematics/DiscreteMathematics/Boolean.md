# 불 대수 (Boolean)
 참과 거짓을 1과 0으로 표현하는 것이다

## 부울 연산자
1. AND 
`A • B`  
A와 B 둘다 1이라면 1  

2. OR  
`A + B`  
A나 B 둘 중 하나라도 1이면 1  

3. NOT  
`~A`  
A가 1이면 0 , 0이면 1

4. XOR  
`A ⊕ B`  
A와 B가 다르면 1

## 부울대수 기본공식

1. 교환법칙  
   `A • B ≡ B • A`  
   `A + B ≡ B + A`  

2. 결합법칙  
   `(A • B) • C ≡ A • (B • C)`  
   `(A + B) + C ≡ A + (B + C)`  

3. 분배법칙  
   `A + (B • C) ≡ (A + B) • (A + C)`  
   `A • (B + C) ≡ (A • B) + (A • C)`  

4. 항등법칙  
   `A • 1 ≡ A`  
   `A + 0 ≡ A`  

5. 지배법칙  
   `A • 0 ≡ 0`  
   `A + 1 ≡ 1`  

6. 부정법칙  
   `~1 ≡ 0`  
   `~0 ≡ 1`  
   `A • (~A) ≡ 0`  
   `A + (~A) ≡ 1`  

7. 이중 부정법칙  
   `~(~A) ≡ A`  
   
8. 멱등법칙  
   `A • A ≡ A`  
   `A + A ≡ A`  

9. 드 모르간 법칙  
   `~(A • B) ≡ (~A) + (~B)`  
   `~(A + B) ≡ (~A) • (~B)`  
   
10. 흡수법칙  
      `A • (A + B) ≡ A`  
      `A + (A • B) ≡ A`  

## 부울함수의 대수적 간소화

복잡한 부울함수를 공식을 이용해 소거하여 간단하게 만드는 것

```
~XYZ + X~YZ + XYZ
= ~XYZ + X~YZ + XYZ + XYZ
= YZ(~X + X) + XZ(~Y + Y)
= YZ + XZ
= Z(YZ)
= XYZ
```
