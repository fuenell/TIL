# Unity Essentials (기초)

### Scene View 조작 법

##### 시선 회전
- `우클릭 드래그`: 시선 회전
- `Alt + 좌클릭 드래그`: 오브젝트 기준 시선 회전

##### 확대
- `휠 스핀`: 확대
- `Alt + 우클릭 드래그 + (Shift)`: 확대 (속도 증가)

##### 이동
- `우클릭 + [W, S, A, D, Q, E] + (Shift)`: 전, 후, 좌, 우, 상, 하 이동 (속도 증가)
- `우클릭 + 휠 스핀`: 이동 속도 조절
- `shift + F`: 선택 오브젝트 위치로 이동
- `Ctrl + Shift + F`: 선택 오브젝트 시점 위치로 이동
- 
##### 도구
- `Q, W, E, R, T, Y`: 도구 변경
- `휠 클릭 드래그`: Hand Tool
- `Ctrl + Alt + 좌클릭 드래그`: Hand Tool
 
===================

- `Ctrl + Shift + C`: 콘솔창 열기
